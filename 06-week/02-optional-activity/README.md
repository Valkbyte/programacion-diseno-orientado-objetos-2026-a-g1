# ¿Qué colocaste en Persona y por qué?
**En esta clase coloqué los atributos que todos tienen en común, por ejemplo documento, nombre y correo, ya que todas las personas los tienen, además, incluí validaciones básicas para evitar datos inválidos, se creó el método ficha() para mostrar la información general.**

# ¿Qué quedó en las subclases y por qué?
*Quedaron dos subclases:*
- La clase Docente hereda de Persona y en adición se coloca el atributo área, porque es algo específico de los docentes, se utilizó "super(...)" para reutilizar el constructor de la clase padre.
- La clase Administrativo también hereda de Persona y agrega el atributo cargo, que es propio de este tipo de personal.
